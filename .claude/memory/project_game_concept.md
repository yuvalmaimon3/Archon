Mobile single-player PvE action roguelite inspired by Archero, built in **Unity 6**.

**Core mechanic:** Virtual joystick movement; auto-fires projectiles only when standing still; targets nearest enemy.

**Structure:** Rooms with enemies → upgrade choice screen → harder rooms → boss encounters.

**Long-term goal:** Online co-op PvE, but current focus is polished single-player foundation.

**Screens:**
- Main Menu
- Gameplay Screen
- Upgrade Choice Screen
- Room Clear / Boss Transition Screen
- Game Over / Results Screen

**Competitor insights:**
- Archero: strong core loop, simple controls, clear progression
- Soul Knight: good enemy variety, fast room-based action
- Survivor.io: strong upgrade satisfaction and power progression

**Avoid:** Excessive grind, forced ads, overloaded UI, overly random upgrade systems.

**Monetization:** Free-to-play, optional rewarded ads, optional non-pay-to-win IAP. Prototype phase = gameplay quality first, no monetization systems.

**Why:** This is the foundational design doc driving all feature and architecture decisions.
**How to apply:** All feature suggestions should align with these pillars. Prioritize gameplay feel, room-based progression, and upgrade satisfaction. Keep systems extensible for future co-op.
